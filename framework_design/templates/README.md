## Usage
All documents created for this project should follow the Qdrant document template format to ensure:
- Easy semantic search
- Consistent structure
- Complete metadata
- Optimal retrieval

## Template Structure
1. Document Title (UPPERCASE)
2. Summary (1-2 sentences)
3. Main Content (organized by sections)
4. Key Points
5. Examples
6. Metadata (for Qdrant payload)
